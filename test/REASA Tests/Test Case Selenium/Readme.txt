Los casos de prueba en este folder estan elaborados con Selenium IDE. Paplicarlos se debe de seguir los siguientes pasos:

-Descargar Selenium IDE utilizando el navegador FireFox: http://seleniumhq.org/download/

-Instalar Selenium como una extension más del navegador.

-Para poder visualizar Selenium IDE hay que activarlo desde la opción Ver -> Panel lateral -> Selenium IDE O bien Herramientas -> Selenium IDE.

-Cargar los test case o test suite desde la opción Archivo -> Open... o Archivo -> Open Test Suite.

-Al estar ya cargados los test case o el test suite presionar el boton verde "Play entire test suite"
